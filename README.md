##如何使用右上角**SETUP**面板？##

**description**：輸入這支API的描述，例如"*登入*"...。

**API path**：API路徑，例如"*http://yourdomain/api/login*"...。(若是絕對路徑，請注意跨網域問題)

**Method**：request方式。

**Add new variable**：新增參數欄位名稱，例如"*Email*"、"*Password*"...。

以上設置完畢，即可按下"*OK,make it.*"按鈕，畫面中會產生你需要的測試面板。

**load**：匯入XML面板資料

**export**：匯出XML面板資料
